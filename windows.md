# Команды Windows
````
netstat -aon                   # Какой процесс слушает порт
netstat -aon | findstr "80"    # Найти кто занимает порт 80
- a                            # показывает все сокеты, используемые процессами
- o                            # показывает PID процесса
- n                            # показывает адреса в числовом формате
````
